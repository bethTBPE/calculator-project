# Простий калькулятор
Навчальний проєкт для вивчення GitHub та Python.
## Опис
Простий консольний калькулятор, який виконує базові арифметичні
операції:
- Додавання (+)
- Віднімання (-)
- Множення (*)
- Ділення (/)
## Запуск
Для запуску програми виконайте команду
python calculator.py
